TYPOGRAPHY

We are using two main fonts, Vollkorn and Rubik. Since these two fonts are font pairing, we combine them with the desire to create a stunning visual effect as well as improve the design's content delivery.


While avoiding the use of display fonts, we prefer to choose two main fonts that are neat, flexible, easy to read, and eye-catching to attract users, and that will be a highlight for identification. trademark. So we chose the above font because it suits the design, style of high aesthetic beauty. Additionally, we'll incorporate custom modifications to these typefaces as needed to make it fresh and different.

Vollkorn

Vollkorn is an Omnibus Type font. The elegance of Vollkorn's capital letters makes it a great option for it suitable for headings. We also use it to make logo wordmark. The font size should be 18 px to 20px for the title. We will resize it if needed.
